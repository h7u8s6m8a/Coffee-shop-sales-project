# Coffee-shop-sales-project:

Coffee shop sales analysis - End-to-end project using SQL ,  python, and power BI to analyze revenue trends, customer behavior, and product performance.

## Project Overview:

The Coffee Shop Sales Analysis Project is an end-to-end data analytics project designed to analyze transactional sales data and generate meaningful business insights.

This project demonstrates the complete workflow of a Data Analyst role — including SQL querying, Python data analysis, and Power BI dashboard visualization.

The objective of this project is to analyze sales performance, customer purchasing patterns, and product-level insights to support data-driven business decisions.

## Business Objectives:

Analyze total revenue, quantity sold, and order trends

Identify top-performing products and categories

Understand sales patterns by month, day, and hour

Compare store performance by location

Detect peak sales hours for better staffing decisions

 ### Tools & Technologies Used:

SQL – Data querying and transformation

Python – Data cleaning and exploratory data analysis

Power BI – Interactive dashboard creation


## Key Analysis Performed:

 ### Sales Performance Analysis:

🔹Total Revenue

🔹Total Orders

🔹Total Quantity Sold

### Time-Based Analysis:

🔹Monthly Sales Trend

🔹Day-wise Sales Performance

🔹Hourly Sales Analysis (Peak Hours Identification)

### Product Analysis:

🔹Top-Selling Products

🔹Revenue by Product Category

🔹Size-wise Sales Distribution

### Store Performance:

🔹Revenue by Store Location

🔹Store Comparison Analysis

## Key Insights:

🔹Identified peak sales hours for operational optimization

🔹Certain product categories contribute significantly to total revenue

🔹Weekend sales are higher compared to weekdays

🔹Specific store locations outperform others in total revenue
